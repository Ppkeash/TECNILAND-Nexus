# 🟢 TECNILAND Nexus

**Launcher de Minecraft moderno y estable para la comunidad TECNILAND.**

TECNILAND Nexus es el launcher oficial de TECNILAND: instala y mantiene los
modpacks de la comunidad al día de forma automática, gestiona Java por ti y te
deja jugar en minutos. Es un fork especializado de
[HeliosLauncher](https://github.com/dscalzi/HeliosLauncher).

## ¿Qué hace?

- **Modpacks siempre al día:** descarga y actualiza solo lo que cambió (Nebula + Cloudflare R2).
- **Cero dolores con Java:** detecta y descarga la versión correcta automáticamente.
- **Cuentas TECNILAND:** login propio con skins personalizadas que funcionan in-game.
- **Cuentas offline:** jugar sin cuenta de Microsoft.
- **Forge estable:** Minecraft 1.13 → 1.21.x.
- **Noticias en vivo:** novedades de TECNILAND desde el panel web.
- **Diagnóstico integrado:** visor de logs en tiempo real con exportación.
- **OptiFine, Discord Rich Presence y UI propia** (tema verde/negro TECNILAND).

## Instalación

Descarga el instalador más reciente desde
[Releases](https://github.com/Ppkeash/TECNILAND-Nexus/releases), ejecútalo y
abre el launcher. Las actualizaciones se aplican solas.
